## Comands of the game

- <i> w </i> - Move Forward;
- <i> s </i> - Move Backwards;
- <i> d </i> - Move Right;
- <i> a </i> - Move Left;
- <i> 1 </i> - Select Material dirt;
- <i> 2 </i> - Select Material glass;
- <i> 3 </i> - Select Material grass;
- <i> 4 </i> - Select Material log;
- <i> 5 </i> - Select Material wood;
- <i> Space </i> - Jump;
- <i> Click </i> - Add Cube;
- <i> Alt + Click </i> - Remove Cube;
